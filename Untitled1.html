Central limit theorem and its implications
Author

Amel Zulji

The Central Limit Theorem (CLT) states that the sampling distribution of the sample mean approaches a normal distribution as the sample size increases, regardless of the underlying population distribution from which the samples are drawn.

To illustrate the CLT, a simulation was performed in which 1 000 independent samples were drawn for each sample size (\(n = 3, 5, 10, \text{and } 20\) ) from each underlying population. For each sample, the mean was calculated, and the distribution of the resulting means were visualised (Figure 1). As the sample size \(n\) increases, the standard deviation of the means (\(\sigma_{\bar{x}}\)) the sampling distribution decreases noticeably, and the distribution converges toward a normal shape. Notably, this is holds true even for the highly right-skewed population (Figure 1 middle column).

Code 
suppressPackageStartupMessages({
  library(tidyverse)
  library(scales)
})

set.seed(4321)

N <- 100000
populations <- tibble(
  Normal = rnorm(N, 0.5, 0.1),
  Uniform = runif(N, 0, 1),
  `Right-Skewed` = rbeta(N, 0.5, 8)
) |>
  pivot_longer(everything(), names_to = "dist_type", values_to = "value")

pop_summary <- populations |>
  group_by(dist_type) |>
  summarise(mu = mean(value), sd = sd(value), .groups = "drop")

pop_list <- split(populations$value, populations$dist_type)
sample_sizes <- c(3, 5, 10, 20)

samples <- expand_grid(dist_type = names(pop_list), n = sample_sizes) |>
  mutate(
    type = paste0("n = ", n),
    value = map2(dist_type, n, \(d, size) {
      replicate(1000, mean(sample(pop_list[[d]], size)))
    })
  ) |>
  unnest(value)

plot_data <- bind_rows(
  populations |> mutate(type = "Population"),
  samples |> select(dist_type, value, type)
) |>
  # order variable so it shows first population followed by increasing n
  mutate(
    type = factor(type, levels = c("Population", paste0("n = ", sample_sizes)))
  )

# define data for spread
rect_data <- plot_data |>
  distinct(dist_type, type) |>
  left_join(pop_summary, by = "dist_type") |>
  mutate(
    n_val = parse_number(as.character(type)),
    spread_label = ifelse(type == "Population", "sigma", "sigma[bar(x)]"),
    width = ifelse(type == "Population", sd, sd / sqrt(n_val)),
    xmin = mu - width,
    xmax = mu + width
  )

# define data for showing centrality
## population centrality
true_mean_data <- pop_summary |>
  select(dist_type, intercept = mu) |>
  mutate(
    line_type_label = "mu",
    type = factor("Population", levels = c("Population", paste0("n = ", sample_sizes)))
  )

## sampling centrality
sim_mean_data <- plot_data |>
  filter(type != "Population") |>
  group_by(dist_type, type) |>
  summarise(intercept = mean(value), .groups = "drop") |>
  mutate(line_type_label = "mu[bar(x)]")

col_pop <- "gray40"
col_sam <- "orange"

Implications

Beyond the convergence of shape, the CLT provides precise predictions regarding the parameters of the sampling distribution. Specifically, it relates the mean and standard deviation of the sample means to the underlying population parameters.

The center of the sampling distribution (\(\mu_{\bar{x}}\)) is equal to the mean of the original population (\(\mu\)).

\(\mu_{\bar{x}} = \mu\)

The variability of the sampling distribution \(\sigma_{\bar{x}}\) (known as the Standard Error of the Mean (S.E.M.)), is directly related to the population standard deviation (\(\sigma\)) and the sample size (\(n\)). As observed in the visual simulation above, the spread decreases as \(n\) increases. This relationship is defined by the inverse-square law:

\(\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}}\)

To validate these theoretical properties, a second simulation was conducted. Three independet samples were drawn from the population with normal distribution and cumulative mean, sd and SEM were computed Samples of increasing size (\(n = 2\) to \(n = 100\)) were drawn from the populations defined previously. For each \(n\), the empirical standard deviation of the sample means was calculated and compared against the theoretical prediction (\(\sigma / \sqrt{n}\)).

To illustrate three independent samples were drawn from a normal population, and cumulative statistics (Mean, SD, and SEM) were calculated as data points were added one by one (\(n=2\) to \(100\)) Figure 2

Code 
n_size <- 100
n_samples <- 3
true_mu <- 0.5
true_sigma <- 0.1

df_cum <- expand_grid(sample_id = paste("Sample", 1:n_samples), n = 1:n_size) |>
  mutate(value = rnorm(n(), true_mu, true_sigma)) |>
  group_by(sample_id) |>
  mutate(
    Mean = cummean(value), # Cumulative mean
    SD = map_dbl(n, \(i) sd(value[1:i])), # cumulative SD
    SEM = SD / sqrt(n) # umulative SE
  ) |>
  filter(n > 1) |>
  ungroup()

# Reshape for plotting
plot_data_cum <- df_cum |>
  pivot_longer(
    cols = c(Mean, SD, SEM),
    names_to = "metric",
    values_to = "observed"
  ) |>
  mutate(
    theoretical = case_when(
      metric == "Mean" ~ true_mu,
      metric == "SD" ~ true_sigma,
      metric == "SEM" ~ true_sigma / sqrt(n)
    ),
    # Create Labels for facets
    metric_label = factor(
      metric,
      levels = c("Mean", "SD", "SEM"),
      labels = c(
        "Cumulative~Mean~(bar(x))",
        "Cumulative~SD~(s)",
        "Cumulative~SEM~(s/sqrt(n))"
      )
    )
  )

# Plot the convergence
ggplot(plot_data_cum, aes(x = n)) +
  geom_point(aes(y = observed, color = sample_id), size = 1.5, alpha = 0.6) +
  geom_line(
    aes(y = theoretical, linetype = "True Parameter"),
    linewidth = 1.2
  ) +

  facet_wrap(
    ~metric_label,
    ncol = 1,
    scales = "free_y",
    labeller = label_parsed
  ) +

  scale_color_brewer(palette = "Set1", name = NULL) +
  scale_linetype_manual(name = NULL, values = c("True Parameter" = "1111")) +

  theme_bw() +
  labs(
    y = NULL,
    x = "Sample Size (n)"
  ) +
  theme(
    legend.position = "bottom",
    legend.text = element_text(size = 15),
    strip.text = element_text(size = 12, face = "bold"),
    strip.background = element_rect(fill = "gray90")
  )
